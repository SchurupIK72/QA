## Идентификаторы кораблей

См. файл `../confidential/ships_ids.json` для полного списка ShipConfigId.

Формат GUID: `20000000-0000-0000-{TYPE}-00000000000{RANK}`

- TYPE: 0001 = Research, 0002 = Cargo, 0003 = Speedboat, 0004 = Warship
- RANK: 0 = нейтральный, 1-4 = боевой ранг
